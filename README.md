# Gank客户端

**有妹子，有视频，有干货的休闲客户端，在你放松的同时，学到新鲜的技术。数据来自代码家的干货网站：http://gank.io**

### 介绍

项目采用MVP架构，同时使用了RxJava2+Retrofit+Glide+GreenDao等第三方库。
